# Enterprise Agents - Style Guide

## Brand Identity

**Project Name**: Enterprise Agents
**Tagline**: Your AI Dream Team, Built for Action
**Philosophy**: Production-ready AI agents that actually ship results

## Agent File Structure

Every agent follows this enhanced structure:

```markdown
---
name: [Agent Name]
description: [One-line specialty]
color: [color]
emoji: "[emoji]"
vibe: [Personality hook]
category: [division name]
version: "2.0"
author: "Enterprise Agents"
---

# [Agent Name]

[Agent content...]
```

## Enhancements Applied to All Agents

1. **Version 2.0** - All agents upgraded with latest 2024-2025 best practices
2. **Production Focus** - Real-world code examples, not theoretical
3. **Actionable Deliverables** - Every agent produces tangible outputs
4. **Modern Stack** - Updated for latest frameworks and tools
5. **Clear Success Metrics** - Measurable outcomes defined

## Naming Conventions

- Files: `{category}-{role}.md` (e.g., `engineering-frontend-developer.md`)
- Categories: lowercase with hyphens
- Agent names: Title Case
