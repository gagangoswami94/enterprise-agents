# Essential Copywriting Frameworks

## AIDA (Attention-Interest-Desire-Action)
Classic framework for any persuasive copy.

### Structure:
1. **Attention**: Hook that stops the scroll
2. **Interest**: Expand on the hook, build curiosity
3. **Desire**: Show transformation, paint the picture
4. **Action**: Clear, compelling CTA

### Example (Email):
```
Subject: Your competitors are using this (and winning)

[ATTENTION]
While you're reading this email, 147 of your competitors just automated their follow-up sequences.

[INTEREST]
Here's what that means: While you're manually sending emails one-by-one, they're nurturing 10x more leads on autopilot.

[DESIRE]
Imagine waking up to 5 new qualified calls booked on your calendar. Every. Single. Day. Without lifting a finger.

That's exactly what happens when you implement what I'm about to show you.

[ACTION]
Click here to see the exact system →
```

---

## PAS (Problem-Agitate-Solution)
Best for awareness-stage content and sales pages.

### Structure:
1. **Problem**: Name their pain clearly
2. **Agitate**: Twist the knife (ethically)
3. **Solution**: Present your answer

### Example (Landing Page):
```
[PROBLEM]
You're drowning in leads but closing almost none of them.

[AGITATE]
Every day that passes, those leads go cold. Your competitors swoop in. The money you spent acquiring them? Wasted.

And the worst part? You know exactly what you should be doing—you just don't have time to do it.

[SOLUTION]
[Product] automatically follows up with every lead, at the perfect time, with the perfect message. So you can focus on closing deals instead of chasing them.
```

---

## Before-After-Bridge (BAB)
Great for transformation-focused messaging.

### Structure:
1. **Before**: Current painful state
2. **After**: Desired future state
3. **Bridge**: How to get there (your product)

### Example (Ad):
```
[BEFORE]
Spending 4 hours a day on manual data entry.

[AFTER]
That same work done in 12 minutes. Automatically.

[BRIDGE]
See how [Product] makes it possible →
```

---

## 4 U's (Urgent, Unique, Useful, Ultra-specific)
Perfect for headlines and subject lines.

### Checklist:
- [ ] URGENT: Is there a reason to act now?
- [ ] UNIQUE: Is this different from alternatives?
- [ ] USEFUL: Does it promise clear value?
- [ ] ULTRA-SPECIFIC: Are there concrete details?

### Examples:
❌ "Grow Your Business" (generic)
✅ "Add 23 New Clients in 30 Days Using This LinkedIn Script"

❌ "Save Money on Software"
✅ "Cut Your SaaS Spend by $2,847/Month (We Did)"

### A/B Testing Framework
```python
# Copy testing and optimization
from dataclasses import dataclass
from typing import Dict, List, Optional
from scipy import stats
import numpy as np

@dataclass
class CopyVariant:
    variant_id: str
    copy_element: str  # headline, cta, body, etc.
    copy_text: str
    impressions: int
    clicks: int
    conversions: int

class CopyOptimizer:
    def __init__(self, confidence_level: float = 0.95):
        self.confidence_level = confidence_level

    def analyze_test(
        self,
        control: CopyVariant,
        treatment: CopyVariant
    ) -> Dict:
        """Analyze A/B test results with statistical significance"""

        # Calculate conversion rates
        control_rate = control.conversions / control.impressions
        treatment_rate = treatment.conversions / treatment.impressions

        # Calculate lift
        lift = (treatment_rate - control_rate) / control_rate

        # Statistical significance test
        contingency = [
            [control.conversions, control.impressions - control.conversions],
            [treatment.conversions, treatment.impressions - treatment.conversions]
        ]
        chi2, p_value, dof, expected = stats.chi2_contingency(contingency)

        is_significant = p_value < (1 - self.confidence_level)

        # Calculate confidence interval for lift
        se = np.sqrt(
            control_rate * (1 - control_rate) / control.impressions +
            treatment_rate * (1 - treatment_rate) / treatment.impressions
        )
        ci_lower = (treatment_rate - control_rate) - 1.96 * se
        ci_upper = (treatment_rate - control_rate) + 1.96 * se

        return {
            'control_rate': f"{control_rate:.2%}",
            'treatment_rate': f"{treatment_rate:.2%}",
            'lift': f"{lift:.1%}",
            'p_value': round(p_value, 4),
            'is_significant': is_significant,
            'confidence_interval': (f"{ci_lower:.2%}", f"{ci_upper:.2%}"),
            'recommendation': self._get_recommendation(
                is_significant, lift, treatment.impressions
            ),
            'winner': 'treatment' if is_significant and lift > 0 else
                      'control' if is_significant and lift < 0 else 'no_winner_yet'
        }

    def _get_recommendation(
        self,
        is_significant: bool,
        lift: float,
        sample_size: int
    ) -> str:
        if is_significant and lift > 0:
            return f"Implement treatment variant - {lift:.1%} improvement is statistically significant"
        elif is_significant and lift < 0:
            return f"Keep control variant - treatment performed {abs(lift):.1%} worse"
        elif sample_size < 1000:
            return "Continue test - need more data for significance"
        else:
            return "Consider ending test - no meaningful difference detected"
```
